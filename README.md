# Amazon-Sales-Analytics-PowerBI
Interactive Amazon Sales Analytics Dashboard Developed using Power BI.
# Project Overview
This project analyzes Amazon sales data using Microsoft Power BI. The dashboard provides interactive visualizations to monitor sales performance, product performance, regional sales, and business insights.
